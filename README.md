# Concurrentia

Concurrentia is a collection of programming exercises focused on concurrency, synchronization, and thread coordination. Each folder contains an independent project that can be used as a learning resource, a teaching example, or a benchmark for studying concurrent behavior.

## Projects

| Folder | Topic | Summary |
| --- | --- | --- |
| `01_A-game-of-NIM-main` | Nim game | A graphical implementation of Nim included mainly for the UI. The gameplay itself is not centered on threading. |
| `02_Arvore_Genealogica` | Family tree | A family tree builder with two implementations: a C++ fork version and a Java thread-based version with images. |
| `03_Problema_Do_Trem` | Train problem | A train management simulation that coordinates concurrent movement and avoids collisions. |
| `04_Problema_do_Produtor_Consumidor` | Producer-consumer | A classic producer-consumer implementation that demonstrates shared-resource synchronization. |
| `05_Jantar_Dos_Filosofos` | Dining philosophers | An implementation of the Dining Philosophers problem that avoids deadlock with a synchronization protocol. |
| `06_Leitores_e_Escritores` | Readers and writers | A shared-resource scenario that allows multiple readers while preventing data corruption during writes. |
| `07_Simulacao_Do_Transito_Automato` | Autonomous traffic simulation | A transit simulation with automated vehicles and stations that uses concurrency to keep the system safe and efficient. |
